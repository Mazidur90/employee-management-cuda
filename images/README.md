# Screenshots

This folder contains screenshots for the README.

To add images:
1. Take screenshots of the running application, architecture diagrams, Docker setup, and CI/CD pipeline.
2. Save them here as dashboard.png, architecture.png, docker-setup.png, ci-cd.png.
3. Commit and push to update the README display.

Placeholders:
- dashboard.png: Screenshot of the simulation dashboard
- architecture.png: Diagram of system architecture
- docker-setup.png: Docker Compose running
- ci-cd.png: GitHub Actions workflow
